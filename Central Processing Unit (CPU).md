It is the electronic circuitry of a computer that carries out the actual processing. Physically, a CPU is placed on one or more microchips called **Integrated Circuits (ICs)**, which are made up of semiconductor materials. As the CPU is located on a microchip, it is also called the microprocessor and is commonly called processor. The CPU is also called the brain of the computer because it is the control centre of the computer where all processing takes place.

The CPU is given instructions and data through programs. It first fetches instructions from memory and then interprets them so as to know what is to be done. If required, data is fetched from memory or input device. Once the information is entered into the computer by the input device, the processor executes or performs the required computation and then stores the result back to memory and displays on the output device.

The CPU has three main components which are responsible for different functions – Arithmetic Logic Unit (ALU), Control Unit (CU) and Memory Registers.

## Arithmetic and Logic Unit

The ALU performs all mathematical calculations and takes logical decisions as per the instructions in a program. Arithmetic operations include addition, subtraction, multiplication, and division. Logical decisions involve comparison of two data items — to check which one is larger, smaller, or whether they are equal. Every calculation done by the computer passes through the ALU.

## Control Unit

The Control Unit is the manager of the CPU. It coordinates and controls the flow of data in and out of the CPU and also controls all the operations of ALU, memory registers and input/output units. It is responsible for carrying out all the instructions stored in the program. It decodes the fetched instruction, interprets (understands) it and then sends control signals to input/output devices until the required operation is done properly by ALU and memory.

## Registers

Registers are temporary units of memory located inside the CPU chip itself. They receive and hold data, instructions, or memory addresses that the CPU is currently working with. They are extremely fast, very small, and limited in size and number. They can be of different sizes — 16-bit, 32-bit, 64-bit, and so on. While processing, each register has a specific function such as storing data, storing an instruction, or storing the address of a location in memory etc. The user registers can be used by an assembly language programmer for storing operands, intermediate results etc. The **Accumulator (ACC)** is the main register in the ALU. It holds one of the operands (values) for any arithmetic or logical operation being performed.
