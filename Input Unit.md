The input unit consists of all the input devices attached to the computer. These devices take raw input from the user and convert it into a digital (binary) form — 0s and 1s — that the computer can understand and process.

Some examples of input devices include keyboard, mouse, scanner, touch screen, microphone etc. Modern computers also support voice input — for example, Google Voice Search allows users to search the web by speaking.

Data entered through input device is temporarily stored in the main memory (also called RAM) of the computer system. For permanent storage and future use, the data as well as instructions are stored permanently in additional storage locations called secondary memory.
