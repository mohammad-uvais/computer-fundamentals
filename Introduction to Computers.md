# Computers in Everyday Life

Computers have now become an indispensable part of our lives so deeply that we interact with them constantly — often without even realizing it. When you use a smartphone, watch something on a smart TV, withdraw money from an ATM, or even heat food in a microwave oven, you are interacting with a computer in some form. Some common examples of computers we come across in everyday life: Desktop, Laptop, Tablet, Smartphone, Smartwatch, Smart TV, Microwave Oven, Washing Machine, Refrigerator, ATM (Automated Teller Machine), TVM (Ticket Vending Machine), POS Machine (Point of Sale), Server Computers, Cloud Computers, and more.

# The Word "Computer" - Literal Meaning

The word **computer** comes from the Latin word _computare_, which means **"to calculate".**

Historically, the word _computer_ did not refer to a machine at all — it referred to a **human being**. Specifically, it was used for a person whose job was to perform mathematical calculations by hand. Astronomers' assistants, wartime navigators, and census workers were all called "computers" because computing — doing calculations — was literally their profession.

The word was only later applied to electronic machines that could automate those calculations far more quickly and accurately than any human could.

So in its most literal sense: **a computer is something (or someone) that computes.**

# What is a Computer?

A computer is an electronic device that accepts **raw data** as input, processes it according to a set of instructions, and produces **meaningful information** as output.

Since the definition itself uses the words _data_ and _information_, it is important to understand what they mean before going further.

**Data** is the term used for raw facts and figures. Data by itself has no meaning. For example, `134`, `+9`, `'Raju'`, `'C'` are all data.

**Information** is data that has been processed and presented in a useful and meaningful form. In simple words we can say that data is the raw material that is processed to give meaningful, ordered or structured information. For example, _"Raju is 9 years old"_ — this conveys a clear meaning and is information about Raju.

The conversion of data into information is called **data processing**, and this is the core purpose of a computer.

# Components of a Computer

A computer is a combination of **hardware** and **software**.

**Hardware** refers to all the physical, electronic components of a computer that we can see and touch — for example, motherboard, central processing unit (CPU), memory devices, storage devices, input device (keyboard), output devices (monitor) etc.

**Software** is the set of programs and instructions that tell the hardware what to do and how to do it. We cannot touch or see software physically.

It is software that gives hardware its capability — hardware is of no use without software, and software cannot run without hardware.

Every task given to a computer follows the **Input → Process → Output (IPO) Cycle**. The computer takes some input, processes it, and produces the desired output. The key components that make this cycle work are:

- **Input Unit** — takes input from the user
- **Central Processing Unit (CPU)** — processes the data
- **Memory** — holds data and instructions during processing
- **Output Unit** — delivers the result to the user
